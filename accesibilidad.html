Declaración de accesibilidad

Compromiso con la accesibilidad

En P&I - Pictures & Images, nos comprometemos a garantizar que nuestro sitio web sea accesible para todas las personas, independientemente de sus capacidades. Continuamente mejoramos la accesibilidad de nuestro sitio para cumplir con las pautas de accesibilidad web y hacer que sea más fácil de usar para todos.

Medidas para apoyar la accesibilidad

Uso de texto alternativo en imágenes y elementos multimedia.

Posibilidad de aumentar el tamaño del texto para mejorar la legibilidad.

Diseño adaptado para dispositivos móviles y diversos tamaños de pantalla.

Opciones de alto contraste para personas con baja visión.

Implementación de un "modo noche" para reducir la fatiga visual.

Compatibilidad con lectores de pantalla y navegadores accesibles.

Características de accesibilidad

Modo noche: Reduce el brillo de la pantalla y ofrece colores oscuros para facilitar la navegación en condiciones de poca luz.

Alto contraste: Aumenta el contraste entre el texto y el fondo para mejorar la visibilidad de los usuarios con baja visión.

Texto agrandado: Proporciona una opción para aumentar el tamaño de la fuente para facilitar la lectura.

Combinación de alto contraste y texto agrandado: Especialmente útil para personas que necesitan ambas adaptaciones.

Compatibilidad con navegadores y tecnologías asistivas

Este sitio web está diseñado para ser compatible con las siguientes tecnologías:

Lectores de pantalla como NVDA y JAWS.

Navegación por teclado y otras tecnologías de asistencia.

Navegadores web modernos como Chrome, Firefox, Safari y Edge.

Estado de conformidad

El sitio web de P&I - Pictures & Images está totalmente conforme con las WCAG 2.1 nivel AA.
Conforme significa que todo el contenido cumple con las pautas de accesibilidad.
